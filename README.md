# デザイン基礎

<h2>文字</h2>
・<strong>質感</strong>を意識することでタイトルがより強調される<br>
・文字を手書き風にする場合は<strong>ラフ効果</strong>を使用する<br>
・フォントの種類でデザインの印象が変わる
・文字に<strong>斜め・歪み効果</strong>を使うことで動きがつく<br>
・スタンプ風の擦れ文字を表現したい場合は<strong>光彩・メゾティント・ラフ効果</strong>を適用する
・数字と英字は<strong>Helveticaフォント</strong>を使用する
・<strgon>助詞</strong>は小さくすることでメリハリが出る
